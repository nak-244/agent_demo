非公開求人について

CATでご紹介できる求人情報の約8割が「非公開求人」です。「非公開求人」とは、採用を進める企業がさまざまな理由から社名を明かさず、人材を募集することです。あなたの希望に合う仕事は、非公開求人の中にこそあるかもしれません。非公開の求人情報のご紹介については、CATにご登録いただくことが必要になります。（お申込後、弊社からご連絡します）

1ステップの簡単手順で登録可能！

総合人材サービスの先駆者のCATだからこそ、企業からご依頼いただける非公開求人の数も多数あります。日ごろの情報収集で満足いく求人情報がなかなかみつからないという方は、自分の可能性を広げるためにも、CATの転職支援サービスをご活用ください。
